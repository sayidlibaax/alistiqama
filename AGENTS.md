# GitHub AI Agent Instructions

## Repo Overview
This repository is a static website for "Al Istiqama Travel Agency" built with plain HTML and CSS.

## Key files
- `index.html` — home page
- `team.html` — team page
- `registration.html` — registration form page
- `contact.html` — contact form page
- `css/style.css` — shared stylesheet

## What to do
- Treat this as a static, client-side website. There is no JavaScript, backend, or build system.
- Make changes by editing HTML and CSS only unless the user explicitly asks to add a script or backend.
- Preserve the existing page structure, branding, and responsive layout.
- Keep navigation links relative and page filenames consistent.
- Use semantic HTML and simple CSS conventions.

## What not to do
- Do not add package managers, npm/yarn tooling, or Node-based build steps.
- Do not assume the repo has tests or automated tooling.
- Do not remove existing responsive/mobile support without a user request.

## Additional notes
- The site is a travel agency landing site with service descriptions, team profiles, and forms.
- Keep styles in `css/style.css` and avoid inline styling unless necessary for small updates.
- If a user asks for optimization or enhancement, prefer clean, accessible HTML/CSS improvements.
